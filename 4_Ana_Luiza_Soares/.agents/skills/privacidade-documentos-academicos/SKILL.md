# Skill: Privacidade em Documentos Acadêmicos

## Regras mínimas

- Não commitar PDFs reais.
- Não armazenar PDF bruto por padrão.
- Não registrar nome, matrícula ou código de verificação.
- Não enviar dados pessoais desnecessários ao LLM.
- Usar fixtures anonimizadas.
- Armazenar apenas dados acadêmicos necessários.
