# Traduções das especificações do EstudaUnB

Esta pasta contém somente traduções em Português do Brasil das especificações canônicas. Ela existe para revisão humana, comunicação acadêmica e acessibilidade; não é a fonte canônica para implementação.

As especificações canônicas estão em [`../specs/`](../specs/README.md). O projeto adotou especificações canônicas em inglês durante o desenvolvimento para padronizar a interpretação por IA e buscar eficiência esperada de tokens no fluxo escolhido. Essa é uma decisão deste projeto, não uma afirmação universal sobre todos os modelos.

Cada tradução deve preservar integralmente o significado técnico, a numeração, os identificadores de requisitos e critérios de aceitação, tabelas, payloads, blocos de código, links, referências e diagramas. Rotas, chaves JSON, variáveis de ambiente, identificadores de código, nomes de banco, valores de enum, arquivos, comandos e IDs Mermaid não são traduzidos.

Em caso de divergência, `../specs/` prevalece. Toda alteração canônica exige atualização da tradução de mesmo nome e revisão dos metadados de sincronização.

## Índice

Consulte o [índice canônico](../specs/README.md#specification-index), que contém status, evidências e relações de evolução. Os arquivos 001 a 018 desta pasta correspondem nominalmente aos arquivos canônicos.
